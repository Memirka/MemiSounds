# Мои чатсаунды

Репозиторий звуков для [neo-chatsounds](https://github.com/Earu/neo-chatsounds).

## Структура

```
sounds/
├── категория/
│   ├── фраза триггер.ogg      → срабатывает на "фраза триггер"
│   └── смех/                  → срабатывает на "смех", играет случайный файл
│       ├── 1.ogg
│       ├── 2.ogg
│       └── !особый смех.ogg   → "!" = триггер по имени файла, а не папки
```

## Правила

- Только **.ogg** (vorbis), **моно**, 44100 Гц — остальное игнорируется
- Имя файла = триггер: нижний регистр, `_` и `-` считаются пробелами
- Названия категорий и файлов — лучше латиницей/транслитом (кириллица в чате
  гмода набирается, но триггеры надёжнее работают в ASCII)
- Репозиторий должен быть **публичным**

## Конвертация

Положи исходники (mp3/wav/webm/что угодно) в папку `raw/` и запусти:

```
./convert.sh raw sounds/моя_категория
```

## Подключение

В `garrysmod/data/chatsounds/repo_config.json` добавь:

```json
{"Branch":"main","BasePath":"sounds","Repo":"ВАШ_НИК/ИМЯ_РЕПОЗИТОРИЯ","UseMsgPack":false}
```

Затем в консоли игры: `chatsounds_recompile`
